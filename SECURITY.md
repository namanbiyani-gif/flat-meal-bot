# Security and privacy

This project uses a linked-device WhatsApp library rather than the official WhatsApp Business API.

Keep the following files private:

- `.env`
- `config/household.json`
- `auth/`
- `data/`
- `logs/`
- `audio/`

Do not include real household names, WhatsApp group IDs, sender IDs, authentication files, databases, or message logs in public issues.

For a security vulnerability, use GitHub's private vulnerability reporting feature when available instead of opening a public issue.
